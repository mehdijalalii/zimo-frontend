# Plan: Create shared PricingCard component

## Goal
Extract the pricing card into a shared component used by both PricingSection (pricing page) and HomePricing (homepage).

## Files
- Create: `src/components/pricing/PricingCard.tsx` — shared card component
- Modify: `src/components/pricing/PricingSection.tsx` — use PricingCard
- Modify: `src/components/home/HomePricing.tsx` — use PricingCard

## PricingCard Props
```tsx
{
  locale: Locale;

  ctaLabel: string;        // "درخواست مشاوره و دمو" or "تماس بگیرید"

}
```
